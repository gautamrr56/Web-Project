Site Report

This portfolio website documents my learning experience in the Web Development module (CSY1063). At the beginning of the term, I had very little understanding of how HTML and CSS work together to create structured and responsive websites. Through weekly practice, experimentation, and debugging, I gradually developed confidence in writing clean and semantic code.

One of the main challenges I faced was creating a consistent layout across all pages while still using different grid structures where required. I overcame this by using a common navigation bar and footer, and applying CSS Grid and Flexbox appropriately for different sections.

The design choices for this site focus on simplicity and readability. I selected neutral background colors with a contrasting highlight color to keep the content clear and accessible. Fonts were kept simple to ensure compatibility across devices.

All HTML and CSS files were validated using the W3C validation tools to ensure compliance with web standards. Screenshots of the validation results are shown below.
